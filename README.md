## Getting started

* Install (first time):
    * `npm install`
* Run:
    * `brunch watch --server` (client)
    * `npm start` (server)
* Test:
    * open your browser at [http://localhost:3333/](http://localhost:3333/)
    * open your REST client at [http://localhost:5000/api](http://localhost:5000/api)
* Learn:
    * go see Wiki
